Git Concepts

Working Directory: The current files you are working on in your project.

Staging Area: A place where changes are added before committing.

Repository: The place where all commits are stored, including the project history.

Changeset: A set of changes made to the files, ready to be staged and committed.

Git Remote and Branching

  1. Initialize a Git repository using git init.
  2. Connect to a remote repository: git remote add origin <url>.
  3. Visualize branches with git branch -v.
  4. Create a new branch: git branch feature-branch.
  5. Switch branches: git checkout feature-branch or git switch feature-branch.
  6. Change the remote name: git remote rename origin new-remote.
